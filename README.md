# Branching-Out Project

This project demonstrates user filtering functionality in Python. It includes a main script and a user filter module.

## Directory Structure
Branching-Out/

├── .venv/             # Virtual environment directory

├── filter_users.py    # Module for filtering users based on specific criteria

├── main.py            # Main script to execute the program

└── users.json          # JSON file containing user data

## Installation

1. Clone the repository:
```bash
   git clone https://github.com/ahmadalshouly/Branching-Out.git
```

2. Set up a virtual environment and install dependencies:
```bash
   python -m venv .venv
   source .venv/bin/activate  # On Windows use `.venv\Scripts\activate`
```

3. Run the main script:
```bash
   python main.py
```

   This will execute the user filtering functionality using the data from `users.json`.

## Development

- `filter_users.py`: Contains the logic for filtering users based on specific criteria.
- `main.py`: The main script that uses the filter functionality.

Feel free to modify the code to suit your specific needs.
